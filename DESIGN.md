# Hermes-Otel Visual Identity

## Style Prompt
Technical developer-focused aesthetic with dark terminal-inspired background, monospace typography for code elements, and electric blue/green accent colors that evoke observability data streams and tracing visualizations. Clean, precise, and information-dense without feeling cluttered.

## Colors
- **Background:** `#0d1117` (GitHub dark dimmed)
- **Surface:** `#161b22` (slightly lighter for cards/panels)
- **Primary accent:** `#58a6ff` (electric blue for highlights, links)
- **Secondary accent:** `#3fb950` (success green for traces/spans)
- **Text primary:** `#c9d1d9` (off-white for headlines)
- **Text secondary:** `#8b949e` (muted gray for body text)

## Typography
- **Headlines:** JetBrains Mono or Fira Code (monospace, 70-120px)
- **Body/labels:** Inter or system-ui (sans-serif, 24-36px)
- **Code snippets:** JetBrains Mono (monospace, 20-28px)

## What NOT to Do
1. Don't use rounded corners larger than 8px — keep it sharp and technical
2. Don't use gradients or soft shadows — flat colors only
3. Don't animate with bouncy eases — use power2/power3 for precision feel
4. Don't exceed 3 accent colors in one frame
5. Don't use serif fonts anywhere
